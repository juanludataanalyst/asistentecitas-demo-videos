## Cómo regenerar la imagen

```bash
# Desde la raíz del proyecto
npx remotion render SaaSPromotionalImage \
  --frames=0 \
  --sequence \
  --image-format=png \
  --output=./promotional-images/saas-dashboard \
  --overwrite
```

## Componente React

Para modificar:

- Colores: Modificar los valores hexadecimales
